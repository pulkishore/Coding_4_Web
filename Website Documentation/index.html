Khoj Studio

Home Page

The homepage is very minimal and uses only b&w and grey theme. The homepage is filled with hover interactions so that the user can view more details without the homepage natively looking cluttered. Whether visiting the site on desktop or mobile, the website remains responsive with animations and interactions and has a mobile breakpoint at 900px. On the navbar we can see 2 main pages- ‘Programmes’ and ‘Thematics’

Index Page

The menu bar/index is simple and has a smooth animation when clicked on. It lists down all pages available to access by the user in one palce.

Pages

The programmes page makes it easy to navigate through available programmes as the user can filter and narrow down whatever programme they want to choose from a vast variety of filters.
